# waterdrop-esphome

## Environment setup

1. Set up esphome through [pip method](https://esphome.io/guides/installing_esphome/#pip)
2. `cd path/to/repo`
2. `ln -s "$HOME/path/to/your/esphome-venv" .venv`

## Building

1. Ctrl+Shift+P
2. `Tasks: Run Task`
3. `ESPHome: Run without debugging`

## Electronics

MCU board: [Waveshare ESP32-S3-Tiny](https://www.waveshare.com/wiki/ESP32-S3-Tiny).
